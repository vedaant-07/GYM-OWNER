# SE7EN FIT Gym Owner Website

Production gym-owner management website for SE7EN FIT.

## Current production backend

This website is connected to the SE7EN FIT Render backend:

```bash
https://se7enfit-original.onrender.com
```

The website also uses Supabase Auth and Supabase tables for the Base44 compatibility adapter.

## Render deployment

This repository includes a Render Blueprint file:

```text
render.yaml
```

Render should create this service:

```text
se7enfit-gym-owner
```

Recommended Render setup:

```text
Project: SE7EN FIT / SE7EN FITS
Service type: Static Site
Repository: vedaant-07/GYM-OWNER
Branch: main
Build command: npm install && npm run build
Publish directory: dist
```

Required environment variables in Render:

```bash
VITE_API_BASE_URL=https://se7enfit-original.onrender.com
VITE_SUPABASE_URL=https://kgzgwapaxgjhmhbphltl.supabase.co
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_publishable_key
NODE_VERSION=22
```

After adding the variables, click **Save, rebuild, and deploy** in Render.

## What was changed from Base44 mode

The original project was generated by Base44. The production version now includes a Render API adapter and a Supabase compatibility adapter so the same UI can run against the live SE7EN FIT stack.

Important files:

```text
src/lib/api-client.js          # Fetch client, JWT storage, JSON/HTML response safety
src/api/base44Client.js        # Compatibility adapter entrypoint
src/api/supabaseBase44Adapter.js # Supabase-backed Base44 compatibility adapter
src/lib/AuthContext.jsx        # JWT/session auth context for owner login/register/session restore
src/pages/Login.jsx            # Owner login
src/pages/Register.jsx         # Owner registration
src/index.css                  # Neon green SE7EN FIT theme
render.yaml                    # Render Static Site deployment config
```

## Notes

- Owner authentication uses Supabase Auth in the compatibility adapter.
- Pages using entity lists are routed through the Supabase adapter. If a table is not built yet, the page receives an empty list instead of crashing.
- Google login on the website is disabled until web OAuth is configured separately.
- Yellow accent colors have been replaced with neon green.
